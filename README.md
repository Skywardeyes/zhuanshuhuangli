# 专属黄历

基于微信云开发的个性化黄历小程序，结合四柱八字命理学与 AI 大模型，为用户提供专属的每日运势指南。

## 功能特性

### 八字命盘
- 分步引导式八字排盘，支持阳历/农历日期输入
- 四柱八字计算：年柱、月柱、日柱、时柱，含天干、地支、十神、藏干
- 大运排盘：起运年龄、大运方向、各步大运干支及年龄区间
- 五行力量分布与旺衰分析
- 神煞识别：天乙贵人、文昌贵人、驿马、桃花等
- 格局判定与喜用神/忌神推导

### 专属黄历
- 日历月视图，每日标注运势评分
- 每日宜忌事项、冲煞提示、五行运势
- 与用户八字的关系分析（干支冲合、十神生克）
- 吉神方位、胎神等传统黄历信息

### AI 智能预生成（DeepSeek V4 Pro）
- 调用外部大模型逐日分析命理，生成个性化宜忌与评分
- 提示词引用《穷通宝典》《三命通会》《滴天髓》《渊海子平》《子平真诠》等经典典籍方法论
- 批量并行调用（5 天/批），支持重试与本地引擎降级
- 预生成进度可视化，按日期范围逐月推进

### 多八字管理
- 支持创建和切换多套八字
- 每套八字独立维护：修改信息、预生成黄历、删除
- 八字切换后全局状态同步（个人信息、黄历数据、主题色）

### 五行主题色
- 根据日主天干自动匹配5套主题色：木（绿）、火（红）、金（灰蓝）、水（蓝）、土（橙）
- 主题切换时全局同步：导航栏、TabBar、页面背景、卡片、按钮、文字全部即时刷新

## 技术架构

```
miniprogram/                 # 小程序前端
├── pages/
│   ├── index/               # 首页 — 今日运势概览
│   ├── huangli/             # 黄历页 — 月视图日历 + 日黄历卡片
│   ├── huangli-detail/      # 黄历详情页 — 完整宜忌/冲合/五行
│   ├── birth-input/         # 八字输入 — 分步向导 + 编辑模式
│   ├── fortune/             # 命盘页 — 四柱表/大运/五行/神煞/简析
│   └── profile/             # 个人中心 — 资料/八字管理/预生成
├── components/
│   ├── bazi-chart/          # 四柱八字表格
│   ├── bazi-switcher/       # 八字下拉切换器
│   ├── calendar-grid/       # 日历月网格
│   ├── dayun-table/         # 大运横向排盘表
│   ├── fortune-indicator/   # 运势评分环
│   ├── huangli-card/        # 黄历卡片（宜忌 + 冲合）
│   └── wuxing-chart/        # 五行力量柱状图
├── utils/
│   ├── api.js               # 云函数调用封装（重试/错误解析）
│   ├── cache.js             # 黄历本地缓存
│   ├── dateUtils.js         # 日期/农历工具
│   └── theme.js             # 五行主题色定义（10天干 × 24色token）
└── app.js                   # 全局状态管理（八字/主题同步）

cloudfunctions/              # 云函数
├── baziFunctions/           # 八字计算、CRUD、排盘
├── huangliFunctions/        # 黄历生成、AI预生成（DeepSeek）、查询
└── userFunctions/           # 用户信息管理
```

### 数据流

```
用户输入出生信息
  → baziFunctions.calculateBazi()
    → bazi-core.js 排盘引擎（四柱/大运/神煞/喜用神）
    → 存储至 bazi_records 集合
  → huangliFunctions.preGenerateHuangli()
    → 构建 Prompt → DeepSeek API（并行批处理）
    → 解析 AI 响应 → 存储至 user_huangli 集合
  → 客户端查询 user_huangli → 渲染日历/卡片/详情
```

## 快速开始

### 环境要求
- 微信开发者工具（基础库 ≥ 2.2.3）
- 已开通微信云开发（云函数 + 数据库）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/Skywardeyes/zhuanshuhuangli.git
   ```

2. **导入微信开发者工具**
   - 打开微信开发者工具 → 导入项目
   - AppID 选择已开通云开发的小程序
   - 项目目录选择本项目根目录

3. **配置云环境**
   - 修改 `miniprogram/app.js` 中的 `env` 为你的云环境 ID
   ```javascript
   env: "your-cloud-env-id"
   ```

4. **创建数据库集合**
   在云开发控制台 → 数据库中创建以下集合：
   | 集合名称 | 权限 | 说明 |
   |----------|------|------|
   | `users` | 仅创建者可读写 | 用户资料 |
   | `bazi_records` | 仅创建者可读写 | 八字记录 |
   | `user_huangli` | 仅创建者可读写 | 黄历缓存 |

   > 建议在 `user_huangli` 集合上创建复合索引：`_openid` + `baziId` + `date`

5. **部署云函数**
   - 在微信开发者工具中，右键 `cloudfunctions/` 下的每个云函数目录
   - 选择「上传并部署：云端安装依赖」

6. **（可选）配置 DeepSeek API Key**
   - 在云开发控制台 → 云函数 → `huangliFunctions` → 编辑环境变量
   - 添加环境变量 `DEEPSEEK_API_KEY`，值为你的 DeepSeek API Key
   - 未配置时将使用本地八字引擎生成，不影响基本功能

### 目录结构快速部署

如需部署单个云函数，请上传该云函数目录（包含 `package.json`），微信云开发会自动安装依赖。

## 环境变量

| 变量名 | 必需 | 说明 |
|--------|------|------|
| `DEEPSEEK_API_KEY` | 否 | DeepSeek API Key，不配置则使用本地引擎 |
